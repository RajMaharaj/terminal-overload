Terminal Overload Server
------------------------

This directory and its subdirectories contain the script files
needed to run a Terminal Overload server.

tol.srv.cs   Server script file for Terminal Overload
common/      Providing common server functionality
base/        Basic game stuff
client/      Interfacing with client script modules
CTF/         Capture the flag game mode
dm/          Deathmatch game mode
eth/         Ethernet game mode
forms/       Player-controllable forms
items/       Various items
shell/       A GUI to start tol server from shell GUI
structures/  Structures with functionality
TE/          Team elimination game mode
weapons/     Player-mounted weapons
