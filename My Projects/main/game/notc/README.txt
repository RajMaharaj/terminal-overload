Terminal Overload Server
------------------------

This directory and its subdirectories contain the script files
needed to run a Terminal Overload server.

tol.srv.cs   Server script file for Terminal Overload
base/        Script files used by all game modes
client/      Script files abstracting interfacing with client script modules
dm/          Script files implementing deathmatch game mode
eth/         Script files implementing ethernet game mode
forms/       Script files implementing player-controllable forms
items/       Script files implementing various items
shell/       Script files implementing a GUI to start tol server from shell GUI
TE/          Script files implementing team elimination game mode
weapons/     Script files implementing player-mounted weapons
